## Introduce
PasteFlow is a powerful clipboard enhancement tool designed exclusively for Mac users. Say goodbye to the traditional single-item clipboard limitation and embrace a world where every copy is saved and instantly accessible.

- Unlimited History
• Automatically save all copied content, support up to 5,000 records
• Full support for text, images, videos, and files
• Smart content type recognition with automatic categorization

- Instant Search
• Full-text search with real-time results
• Smart grouping by content type or source application
• Pin frequently used items for quick access

- Efficient Operations
• Customizable hotkeys for instant panel access
• Source app icon identification for easy tracking
• Precise timestamp for every entry

- Beautiful Interface
• Elegant vibrancy design that blends seamlessly with macOS
• Dark/Light themes with automatic system switching
• Center/Bottom display modes for flexible positioning
• Customizable corner radius, font size, and more

- Privacy & Security
• All data stored locally, never uploaded to cloud
• Automatic filtering of sensitive content

- Perfect For
• Writing & Content Creation - Access historical text snippets instantly
• Programming - Quickly reuse code fragments
• Design Work - Manage your image asset library
• Daily Office Work - Boost document editing efficiency
• Multitasking - Seamlessly transfer information between apps

Let PasteFlow be your productivity multiplier and enter the era of intelligent clipboard management!
